github连不上？下载龟速浪费生命？这些方法一定得看看！

有时候想要上github找点东西，或者看到一个好玩的项目结果作者给了一个github链接。开兴的点开链接却发现打不开，靠北啊！这时候就需要一些“小技巧”啦！

[github快速访问小技巧]

了解需求

首先你需要了解自己的需求

  • 刚好看到一条github链接，只需要访问下载! 请看轻度使用
  • 需要登录账号或者使用git命令! 请看更多需求
  • 我能打开github，但下载太慢了！ 请看下载加速

轻度使用

轻度使用

如果你只需要轻度使用，偶尔打开github啦，偶然间看到开源项目想下载啦，只需使用镜像链接即可。只需要在链接中的github，前面加上一个k，变成kgithub就能轻松访问镜像链接啦！
示例：

ps:请勿在镜像站登录账号

更多需求

更多需求

如果你有登录账号、使用git命令等需求，魔法是最稳定的没有魔法或者不方便使用魔法的时候，可以试试这款开源软件：

Fetch GitHub Hosts

支持版本：Windows/Linux/MacOS
项目原理：此项目是通过部署此项目本身的服务器来获取github.com的 hosts，而不是通过第三方ip地址接口来进行获取，例如ipaddress.com等。

软件提供了图形化界面，运行对应平台的执行文件即可运行。

下载加速

下载加速

在github下载太慢了？开以试试这款油猴插件！

  • 为 Github 的 Git Clone/SSH、Release、Raw、Code(ZIP) 等任何需要下载的文件添加 高速下载（加速下载）！
  • 在项目列表页，添加单文件快捷下载 (☁) 链接，方便单独下载某个文件！
  • 在项目 Git 地址处，自动在开头添加 git clone 命令，默认开启！（下图不是最新因此未显示）

如何安装？

如果安装过油猴插件，只需要点击链接！，打开插件界面，点击安装即可！

如果没有安装油猴插件，请看这篇文章进行安装！
